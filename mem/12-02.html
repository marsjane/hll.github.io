Training kit

This training kit is prepared for DTL newcomers (without computer science background) to pick up the programming skill required for their daily work. You are recommended to go through all the following materials. Make sure you are familiar with all the topics mentioned in the requirement of each topic (Python, C++, Bash, Git). You are also encouraged to learn from other material based on your skill level.

Note that this is very important and there will be tests on the required topics when you are on baord.

Python (Optional for researchers)

  1. Requirement:
    • Basic usage and manipulation of data structures: list, set, series, dictionary, numpy array, dataframe.
    • Conditional and iterative statement.
    • Function.
    • Knowledge in machine learning libraries is a plus. (Optional)
  2. If you do not have prior experience in Python programming, please register a free codeacademy account and take this online course on basic Python programming. (Estimated time: 2 days)
    https://www.codecademy.com/learn/learn-python
  3. Check out and go through the Python Basic Tutorials jupyter notebooks (Estimated time: 1 day)
  4. Complete the Python Basic Tutorial Exercises without referring to the tutorials (Estimated time: 1 day)
  5. If you would like to learn more about Python, take this serious online course on Coursera: https://www.coursera.org/specializations/data-science-python. (Optional)
  6. Two ebooks are provided for your reference.
    • Python for Data Analysis, by Wes McKinney
    • Hands on Machine Learning with Scikit Learn and TensorFlow, by by Aurélien Géron

C++

  1. Requirement:
    • Basic C/C++ concepts.
    • Frequently used utilities/data structures: string, vector, stack, map/unordered map/set, list, queue/deque.
    • Object-oriented programming: struct & class (access control, constructor, inheritance, overload, polymorphism, virtual).
    • Pointer and pointer operations, by value/by reference/smart pointer.
  2. If you do not have prior experience in C++ programming, please take this online course on Udacity.
    https://www.udacity.com/course/c-for-programmers--ud210
  3. Go through this tutorial for the basics: http://www.cplusplus.com/doc/tutorial/
  4. Read the following chapters of C++ Primer, Fifth Edition as a reference.
    • Part I: The Basics is a must.
    • The rest chapters are optional.
    • Focus on topics of Standard Template Library, Multi-threading, and OOP.

Bash

  1. Required commands:
    • Basic commands: ls, mkdir, cp, mv, rm, ps, top, kill, CTRL-C/CTRL-Z/CTRL-R.
    • File operations: cat/more/less, sort, comm, diff, cut, tr, awk, sed.
    • Usage of terminal multiplexer: screen.
  2. If you do not have any prior experience in Linux and Bash, please take the following online course on codeacademy. (Estimated time: 5 hrs)
    https://www.codecademy.com/learn/learn-the-command-line
  3. Recap the knowledge by going through this Guide: http://tldp.org/LDP/Bash-Beginners-Guide/html/ (Estimated time: 1 day)

Git

  1. Required git functions: git add, commit, push, pull, branch, fetch, merge.
  2. If you have no prior experience in using Git, please take the following codeacademy online course. (Estimated time: 4 hrs)
    https://www.codecademy.com/learn/learn-git
  3. Take this official guideline document as the reference: https://git-scm.com/book/en/v2

Vim

You will be using Vim as the text editor for your daily research work.

  1. Take this interactive tutorial for the basic usage of Vim: https://www.openvim.com/
  2. Vim is installed with macOS and ubuntu system. If you are using Windows, please learn how to install Vim on your own.
  3. Please use Vim as the text editor when you work on the above Python and C++ sections and practice the vim arguments you learn.
  4. Take the following Vim guideline as a reference.
    http://www.yolinux.com/TUTORIALS/LinuxTutorialAdvanced_vi.html
